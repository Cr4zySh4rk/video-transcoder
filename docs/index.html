Get Started

VideoForge runs a tiny local server on your machine. Download the launcher for your OS, run it once, and come back here to transcode.

1
Install Node.js from nodejs.org (LTS recommended)
2
Install FFmpeg: winget install Gyan.FFmpeg or download from ffmpeg.org and add to PATH
3
Download and double-click the launcher — installs deps on first run and opens this page:
4
Keep the console window open while transcoding. Press Ctrl+C to stop.
1
Install Node.js: brew install node
2
Install FFmpeg: brew install ffmpeg
3
Download the launcher and run it in Terminal:

chmod +x launch-mac.sh && ./launch-mac.sh
4
Keep the Terminal window open while transcoding.
1
Install Node.js: sudo apt install nodejs npm
2
Install FFmpeg: sudo apt install ffmpeg
3
Download the launcher and run it:

chmod +x launch-linux.sh && ./launch-linux.sh
4
Keep the terminal open while transcoding.
1
Download the VideoForge Android APK from the GitHub Releases page and install it on your Android device.
2
On your PC or Mac, run the VideoForge server (Windows/macOS/Linux launcher above, or the desktop app).
3
Make sure your Android device and PC are on the same Wi-Fi network. Find your PC's local IP address (e.g. 192.168.1.5).
4
Open the Android app, tap Connect to Server, and enter http://192.168.1.5:3000 (replace with your PC’s IP).
5
You can now control transcoding jobs from your Android device. The video file is processed on your PC — Android is a remote control interface.

VideoForge

A free, open-source video transcoder that runs entirely on your own hardware. No cloud uploads. No limits.

cr4zysh4rk.github.io/video-transcoder
Features
  • H.264, H.265, VP8/VP9, AV1 codecs
  • MP4, MKV, WebM, MOV containers
  • CRF quality or two-pass CBR bitrate
  • NVIDIA, AMD, Intel & Apple GPU acceleration
  • Dialogue / background audio level control
  • Subtitle passthrough, trim, FPS control
  • Batch transcode, real-time progress via WebSocket
