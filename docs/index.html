Get Started

VideoForge runs a tiny local server on your machine — this page connects to it and provides the interface. Download the launcher for your OS, run it once, and you're set.

1
Install Node.js from nodejs.org (LTS recommended)
2
Install FFmpeg — download from ffmpeg.org and add to PATH, or run: winget install Gyan.FFmpeg
3
Download and double-click the launcher — it installs dependencies on first run and opens this page automatically:
4
Keep the console window open while transcoding. Press Ctrl+C to stop.
1
Install Node.js: brew install node
2
Install FFmpeg: brew install ffmpeg
3
Download the launcher, then run it in Terminal:

chmod +x launch-mac.sh && ./launch-mac.sh
4
Keep the Terminal window open while transcoding.
