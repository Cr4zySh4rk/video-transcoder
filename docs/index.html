Get Started

VideoForge runs a tiny local server on your machine. Download the launcher for your OS, run it once, and come back here to transcode.

1
Install Node.js from nodejs.org (LTS recommended)
2
Install FFmpeg: winget install Gyan.FFmpeg or download from ffmpeg.org and add to PATH
3
Download and double-click the launcher — installs deps on first run and opens this page:
4
Keep the console window open while transcoding. Press Ctrl+C to stop.
1
Install Node.js: brew install node
2
Install FFmpeg: brew install ffmpeg
3
Download the launcher and run it in Terminal:

chmod +x launch-mac.sh && ./launch-mac.sh
4
Keep the Terminal window open while transcoding.
1

2

chmod +x launch-linux.sh && ./launch-linux.sh
4
Keep the terminal open while transcoding.
